# Palindrome Checker 🔄

A simple Java application that checks whether a given word, phrase, or number is a palindrome.

## 📖 What is a Palindrome?

A palindrome is a sequence that reads the same forward and backward.

### Examples

| Input | Output |
|---------|---------|
| madam | Palindrome |
| racecar | Palindrome |
| level | Palindrome |
| hello | Not a Palindrome |

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/PalinCheck.git
```

### 2. Navigate to the Project Directory

```bash
cd PalinCheck
```

### 3. Compile the Java File

```bash
javac PalindromeChecker.java
```

### 4. Run the Program

```bash
java PalindromeChecker
```
